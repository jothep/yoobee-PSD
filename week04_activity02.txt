This system supports core college workflows. 
Students can register, enroll in subjects, submit assignments and view grades and feedback.
Lecturers can generate subject, grade students’ assignments, grade students’ submissions and give feedback to students.
Administrators manage the system of user profiles, class and subjects, users' permission, and reset passowrd for all users.

the number of actors is 3
student

    register
    login/logout
    enroll in subject 
    submit assignments
    view grades and feedback
    change password

lecturer 
- 
    login/logout
    create subject
    grade students' assigments
    give feedback to student
    change password

- 
    login/logout
    manage class and subjects
    manage users, include grant permission and delete user
