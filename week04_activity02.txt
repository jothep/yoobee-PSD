This system supports YB college workflows. 
Students can register, enroll in subjects, submit assignments, and view grades and feedback.
Lecturers can manage class, publish assigments, grade student submissions, and give feedback.
Administrators manage user profiles, classes and subjects, user permissions, and can reset passowrd for all users.

Student 
-   
    Register
    Login/Logout
    Enroll in subject 
    Submit assignments
    View grades and feedback
    Change password

Lecturer 
- 
    Login/Logout
    Create/Manage subject
    Publish/Manage assignments
    Grade student submissions
    Change password
     
Admin
- 
    Login/Logout
    Manage users, setup class info, create or disable account, grant and revoke permissions, delete users
    Reset password for all users
